### Objective
Analyze how different customer segments contribute to order volume, order value, and profitability.

### Work Done
- Segmented customers into High, Medium, and Low Value groups
- Calculated total orders by customer segment
- Computed Average Order Value and Average Profit per Order
- Compared volume-driven vs value-driven customer behavior
- Created visual charts to highlight differences

### Key Insights
- Low Value customers drive the highest order volume but lowest profitability
- High Value customers generate fewer orders but higher profit per order
- Medium Value customers show strong upsell potential

### Business Value
- Helps prioritize customer retention strategies
- Supports efficient marketing spend
- Improves profitability-focused decision making
